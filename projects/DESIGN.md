# DESIGN.md

## 气质与意象

关键词：旅途中的晨光、推开窗的第一口空气、地图上指尖划过的路线

具象场景：清晨六点的火车站候车厅，大落地窗外天色微亮，手里握着一杯热饮和一张刚刚打印出来的行程单。不拥挤、不焦虑，而是带着期待和确定感。产品应该像那个帮你把混乱信息整理成一张清晰行程单的旅伴。

## 视觉策略

- 图像方向：真实的中国风景摄影，偏暖调、有空气感（晨雾、夕阳、雨后），避免过度饱和和滤镜感
- 图形语言：圆角卡片 + 轻微阴影，模拟"摊开的旅行手册"的质感；路线/时间轴用细线条串联，像手绘地图上的标注

## 配色方案

- 主色：暖石灰 `#6B7280` — 像清晨天空的灰调，沉稳不抢戏
- 强调色：晨曦橙 `#F97316` — 像日出第一缕光，用于CTA和关键信息高亮
- 背景：米白 `#FAFAF9` — 像纸张的底色，温暖干净
- 卡片：纯白 `#FFFFFF` — 干净的承载面
- 文字：深炭 `#1C1917` — 高对比可读性
- 辅助：季节色（春绿 `#22C55E`、夏蓝 `#3B82F6`、秋金 `#EAB308`、冬银 `#94A3B8`）用于时间/季节标签
- 警示：柔红 `#EF4444` — 用于风险提示（高反、封山等）

## 字体排版

- 字体族：思源黑体 / Noto Sans SC（Google Fonts CN），中文优先的清晰无衬线
- 标题：18-24px / font-weight 600，紧凑行高 1.3
- 正文：14-16px / font-weight 400，行高 1.6 确保长文可读
- 标签/辅助：12px / font-weight 500，用于天数、预算等小标签
- 排版节奏：问答阶段居中、宽松；方案展示阶段左对齐、信息密度适中

## 动效与交互

- 运动性格：从容、流畅，像翻页而非跳转
- 页面过渡：淡入 + 轻微上移（translateY 8px → 0），200ms ease-out
- 问答步骤切换：当前步骤淡出 → 下一步淡入，不打断心流
- 条件调整重算：卡片内容轻微闪烁过渡，不是硬替换
- 悬停：卡片微升（translateY -2px）+ 阴影加深
- 流式输出：打字机效果，逐字呈现

## 页面结构

### 问答引导阶段
- 居中布局，最大宽度 640px
- 顶部进度条（细线 + 步骤点）
- 问题文字大号居中
- 图片卡片 2x2 网格 + 下方文字输入框
- 底部"下一步"按钮

### 方案展示阶段
- 左侧：条件调整面板（可折叠）
- 右侧：方案卡片纵向排列
- 每个方案卡片：目的地名 + 标签 + 天数概览 → 展开显示逐日行程
- 底部聊天区域：可展开的对话面板

## 设计禁忌

- 不要用科技蓝/紫渐变 — 这是旅行产品，不是SaaS
- 不要用纯黑背景 — 压抑，与"出发"的情绪矛盾
- 不要堆砌信息 — 问答阶段每屏只做一件事
- 不要用廉价旅行网站常用的"特价""限时"等紧迫感文案
- 不要用emoji做图标 — 用 Lucide Icons
- 方案之间必须有明确差异，避免"三个方案本质一样"
